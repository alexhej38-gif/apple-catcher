# Apple Catcher

Простая игра на Python и Pygame, где нужно ловить падающие яблоки.

## Особенности
* Система рекордов (сохраняются в файл).
* Быстрый рывок (dash) по двойному нажатию стрелок.
* Пиксельная графика.

## Как запустить
1. Установите Python.
2. Установите библиотеку Pygame:
   ```bash
   pip install pygame
   ```
3. Запустите игру:
   ```bash
   python main.py
   ```

## Управление
* **Стрелки влево/вправо** — движение.
* **Двойное нажатие стрелки** — быстрый рывок.
* **1 / 2** — выбор в меню.
